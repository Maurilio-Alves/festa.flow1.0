# 🎈 Festa-Flow 1.0

Sistema de gestão inteligente para a **Grazi Decorações**, focado em organizar orçamentos, estoques, agenda de eventos e controle de equipe.

## 🚀 Funcionalidades

- **Gestão de Clientes:** Cadastro completo e histórico de eventos.
- **Orçamentos:** Geração automática de orçamentos e Ordens de Serviço (PDF).
- **Estoque de Acessórios:** Controle total de itens disponíveis para locação.
- **Agenda:** Gerenciamento de eventos e compromissos internos.
- **Segurança:** Sistema de Login e Auditoria de ações (Logs).

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Streamlit** (Interface Web)
- **SQLite3** (Banco de dados)
- **FPDF** (Geração de relatórios)
